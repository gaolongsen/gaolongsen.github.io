While, it’s kind of C++ lib problem and I found the best solution for me to deal with the problem:

  1. First, uninstall the yaml lib from your PC:

    sudo rm /usr/local/lib/libyaml-cpp*
sudo rm /lib/x86_64-linux-gnu/libyaml-cpp*

  2. Download the newest version of libyaml-cpp :

    git clone https://github.com/jbeder/yaml-cpp.git

  3. Then go the the folder you downloaded just now:

   cd ~/yaml-cpp

   mkdir build

   cd build

   cmake ..

   make

   make install

  1. Then close your terminal and go to your catkin_ur folder and then run:

    catkin_make clean

    rm -rf build devel

    catkin_make

After that you will see the that even though there are some warnings shown in the terminal, but you can compile the library successfully with 0 error!
